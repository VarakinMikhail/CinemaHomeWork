Программа обладает понятным интерфейсом (она сообщает пользователю варианты действий и отображает всю необходимую информацию), поэтому пользоваться ей просто. 
Программа сохраняет информацию в файлы формата .json, используя сериализацию. 
Изначальный размер кинозала определен заранее, поскольку, согласно условию, кинозал один, а значит, он заранее определен.
Для авторизации в качестве администратора необходимо использовать пароль: HSE (три заглавные буквы)
В качестве бонуса есть присутствует авторизация и регистрация пользователей (зрителей), то есть данное приложение не только для администратора, но и для зрителей, которые могут купить или сдать билет
Пароли пользователей (зрителей) не хранятся, для авторизации используется специальная функция, найти обратную к которой не представляется возможным (для этого я использовал операции с многочленами высоких степеней и приближением синуса с помощью локальной формулы Тейлора, после которых не представляется возможным найти исходное значение (в том числе, из-за того, что возникают переполнения, из-за чего результатом функции является не значение многочлена в точке, а некоторое значение, не имеющее к нему отношение, поэтому единственным возможным вариантом отгадать пароль является полный перебор (но это тоже осложняется тем, что пароли имеют минимальную допустимую длину)
